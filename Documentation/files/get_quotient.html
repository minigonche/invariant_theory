Invariant Rings > getQuotientRing

getQuotientRing(varList, polinomialList)

Function that given a set of linearly independet polinomials f1,..,fm in the variables x_1,..,x_n, computes QQ[f1,..,fm]

Synopsis

  • Usage:
    getQuotientRing(v, p)
  • Function: getQuotientRing
  • Inputs:
    • v, list of variables of the polynomials
    • p, a list of polynomials
  • Outputs:
    • The corresponing ring of polynomials.
