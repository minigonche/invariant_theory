Invariant Rings > reynoldsOperator

reynoldsOperator( f, varList, matrixList )

Function that calculates the reynolds operator for a given polinomial and matrix list.

Synopsis

  • Usage:
    reynoldsOperator(f,v,m)
  • Function: reynoldsOperator
  • Inputs:
    • f, a polynomial
    • v, list of variables of the polynomial
    • m, a list of matrices, must be over QQ
  • Outputs:
    • The corresponing polynomial, invariant under the action of the given matrices.

Description

Given a list of matrices m, for example S2, and poynomial, the method gives an invariant polynomial under the action of this matrices. 
i1 : R = QQ[x_1,x_2];

i2 : s2 = getSn(2);

i3 : f = 3*x_1^2 + x_2 ;




i4 : reynoldsOperator(f,gens(R),s2)
