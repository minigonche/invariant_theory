Invariant Rings > getInvariantRepresentation

getInvariantRepresentation( matrixList )

Function that given a list of matrices, calculates the invariant ring under their action.

Synopsis

  • Usage:
    getInvariantRepresentation(m)
  • Function: getInvariantRepresentation
  • Inputs:
    • m, a list of matrices, must be over QQ
  • Outputs:
    • A ring of invariants

Description

Function that returns the ring of invariants given a matrixlist representing the action of a given finite group under a representation. 

i1 : s3 = getSn(3);




i2 : invRing = getInvariantRepresentation(s3);
