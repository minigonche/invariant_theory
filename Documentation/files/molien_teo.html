molienTeo( matrixList )

Calculates the Hilbert series of a given matrix list

Synopsis

  • Usage:
    molienTeo(m)
  • Function: molienTeo
  • Inputs:
    • m, a list of matrices, must be over QQ
  • Outputs:
    • A tuple
      • The hilbert series as a fraction field The variabe t used in the series.

Description

Given a list of matrices m, for example S2, the method outputs the corresponding Hilbert Series and variable: 

i1 : s2 = getSn(2)

o1 = {| 1 0 |, | 0 1 |}
      | 0 1 |  | 1 0 |

o1 : List



i2 : molienTeo(s2)

               1
o2 = {---------------, t}
         3    2
        t  - t  - t + 1

o2 : List
